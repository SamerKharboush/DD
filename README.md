## How It Works

### 1. Query Flow

```
User Query вҶ’ Agent Runner вҶ’ Hybrid Router вҶ’ Multi-Agent Orchestrator
                                                           вҶ“
                         в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
                         в”Ӯ                           Specialist Agents                        в”Ӯ
                         в”Ӯ  ChemistAgent вҶ’ BiologistAgent вҶ’ ToxicologistAgent вҶ’ StatisticianAgent
                         в”Ӯ                                                                   в”Ӯ
                         в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
                                                           вҶ“
                                                    Critic Agent (Review)
                                                           вҶ“
                                                    Tool Execution
                                                           вҶ“
                                                    Final Response
```

### 2. Multi-Agent Orchestration

The system uses the **Robin paradigm** with adversarial critique:

| Agent | Role | Expertise |
|-------|------|-----------|
| **ChemistAgent** | Molecular design | Drug-likeness, SAR, synthesis |
| **BiologistAgent** | Target biology | Pathways, MOA, biomarkers |
| **ToxicologistAgent** | Safety assessment | hERG, DILI, mutagenicity |
| **StatisticianAgent** | Data validation | Power analysis, significance |
| **Critic Agent** | Adversarial review | Challenges assumptions |

### 3. Routing Logic

The Hybrid Router intelligently selects between local and cloud models:

| Query Type | Route | Cost |
|------------|-------|------|
| Simple lookup | Local 7B | $0.10/1M tokens |
| Standard analysis | Local 70B | $0.50/1M tokens |
| Complex design | Cloud Sonnet | $3.00/1M tokens |
| Expert reasoning | Cloud Opus | $15.00/1M tokens |
